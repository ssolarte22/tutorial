Responsive Design (Media Queries)

1) ¿Qué es el Diseño Responsivo?

El diseño responsivo asegura que tu página web se vea bien y sea funcional en todos los dispositivos (teléfonos móviles, tablets, pantallas de escritorio). Esto se logra evitando anchos fijos (como width: 800px;) a favor de porcentajes (width: 100%;) y utilizando Media Queries.

Además, es obligatorio incluir la etiqueta meta de viewport en el <head> de tu documento:
<meta name="viewport" content="width=device-width, initial-scale=1.0">

2) Media Queries (@media)

Una media query permite aplicar reglas CSS solamente si el navegador cumple cierta condición, generalmente el ancho de la pantalla (max-width o min-width). 

/* Si el ancho de la pantalla es menor o igual a 768px: */
@media (max-width: 768px) {
  body {
    background-color: lightblue;
  }
}

4) Uso de max-width en imágenes

Para evitar que una imagen se salga de su contenedor en pantallas pequeñas, la regla de oro es aplicarle: max-width: 100%; height: auto;
