Imágenes vectoriales asombrosas (SVG)

1) ¿Qué es SVG?

SVG (Scalable Vector Graphics) es un formato de imagen vectorial basado en XML. A diferencia de los formatos de píxeles (como JPG o PNG), un SVG no pierde calidad al ampliarse, pesa muy poco si es simple, y puede ser manipulado directamente con CSS y JavaScript.

2) Formas básicas

Puedes dibujar formas directamente escribiendo código HTML dentro de la etiqueta <svg>. Las más comunes son <rect> (rectángulo), <circle> (círculo), y <polygon>.

3) Manipulación con CSS

Un aspecto poderoso del SVG en línea es que responde a CSS. Puedes modificar atributos como fill (color de fondo) y stroke (color de borde) en tu archivo .css o al pasar el ratón por encima (hover).

Pasa el ratón por encima del triángulo.

4) Uso práctico

  • Iconos: Los SVG son el estándar actual para pintar iconos (ej. FontAwesome, Material Icons).
  • Logos: Mantienen la identidad de marca nítida tanto en un móvil pequeño como en una pantalla 4K.
  • Animaciones vectoriales: Puedes combinar SVG y las @keyframes de CSS para animar contornos y figuras de forma fluida.
