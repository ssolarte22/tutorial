# Práctica de Arquitectura Frontend y Backend

Este proyecto contiene los ejercicios y tareas desarrolladas durante el curso. A continuación se explica la organización del proyecto:

## 1. Parte del Tutorial (HTML y CSS)

Los archivos que se encuentran en la raíz del proyecto (junto con las carpetas `css/` e `imagenes/`) conforman la **parte del tutorial**. Esta sección incluye múltiples archivos HTML (`animacion.html`, `posicion.html`, `grid.html`, etc.) donde se explican y se ponen en práctica diferentes conceptos de HTML5 y CSS3.

## 2. Parte de Bootstrap

La carpeta **`Arquitectura frontend y backend/`** corresponde a la **parte de Bootstrap** de la tarea. Aquí se encuentra el código que utiliza el framework Bootstrap para la maquetación y diseño de la interfaz web responsiva.
