Insertar contenido externo con iframe

1) ¿Qué es la etiqueta <iframe>?

Un iframe (Inline Frame) es un documento HTML incrustado dentro de otro documento HTML. Se utiliza comunmente para mostrar contenido de otros sitios web sin que el usuario tenga que salir de tu pagina. Los usos mas populares incluyen incrustar mapas, videos de YouTube o widgets de redes sociales.

2) Atributos importantes

  • src: La URL del contenido que quieres incrustar.
  • width y height: Dimensiones del frame (se recomienda controlarlo desde CSS).
  • title: Descriptivo para la accesibilidad (muy importante para lectores de pantalla).
  • allowfullscreen: Permite que el contenido (como un video) se ponga en pantalla completa.
  • loading="lazy": Hace que el iframe solo se cargue cuando el usuario se acerque a el haciendo scroll (mejora rendimiento).

3) Demo: Incrustar un Mapa de Google Maps

Puedes buscar cualquier lugar en Google Maps, presionar "Compartir" y elegir "Insertar un mapa" para obtener el codigo iframe que ves aqui abajo.

4) Demo: Incrustar un Video de YouTube

Similar a los mapas, en cualquier video de YouTube puedes dar "Compartir" -> "Insertar" para obtener su <iframe>. Para que sea responsive (que se adapte al movil manteniendo su ratio de aspecto) se suele usar un contenedor CSS especifico.
