Centrar elementos en la pantalla (Posicionamiento)

Centrar cajas o bloques (<div>) ha sido históricamente uno de los mayores dolores de cabeza en CSS. Por suerte, hoy en día disponemos de múltiples herramientas y estrategias muy sencillas para ello.

1. El clásico: margin: 0 auto;

Si tu elemento es un bloque (display: block) y tiene un ancho fijo (width), puedes usar los márgenes automáticos para centrarlo horizontalmente.

Centrado clásico (Margin)

2. El rey moderno: Flexbox

Añadiendo display: flex; justify-content: center; align-items: center; al contenedor, el elemento hijo queda centrado perfecta y automáticamente sin importar su tamaño.

Centrado Total con Flexbox

3. El más rápido: CSS Grid

Con Grid puedes lograr el mismo efecto de una forma aún más corta usando sólo la regla place-items: center; en el contenedor padre.

Centrado Total con Grid

4. Posicionamiento Absoluto y Transformaciones

Útil para elementos flotantes (pop-ups, modales). Usamos un contenedor con position: relative y el hijo con position: absolute; top: 50%; left: 50%;, ajustando su posición con transform: translate(-50%, -50%);.

Centrado Flotante (Absoluto)
