Animaciones con CSS (animation)

1) Que es una animacion en CSS

Una animacion en CSS permite cambiar el estado visual de un elemento sin depender de JavaScript. A diferencia de transition, la animacion puede reproducirse automaticamente al cargar la pagina y puede repetirse indefinidamente.

Las dos piezas clave son:

  • @keyframes: define los pasos del movimiento o efecto.
  • animation: aplica esos pasos a un elemento concreto.

2) Sintaxis basica

@keyframes aparecer {
  from { opacity: 0; transform: translateY(20px); }
  to   { opacity: 1; transform: translateY(0); }
}

.caja {
  animation: aparecer 1s ease-out 0s 1 normal forwards;
}

Orden habitual de la propiedad animation: nombre duracion timing-function delay iteraciones direccion fill-mode.

6) Propiedades mas importantes

  • animation-name: nombre de la secuencia @keyframes.
  • animation-duration: tiempo total de una vuelta.
  • animation-timing-function: velocidad (linear, ease, ease-in-out...).
  • animation-delay: espera antes de iniciar.
  • animation-iteration-count: numero de repeticiones o infinite.
  • animation-direction: normal, reverse, alternate, alternate-reverse.
  • animation-fill-mode: controla estilos antes/despues (none, forwards, backwards, both).

7) Recomendaciones practicas

  • Anima preferentemente transform y opacity por rendimiento.
  • Evita animaciones demasiado largas o agresivas en interfaces de lectura.
  • Usa duraciones coherentes: entre 200ms y 1200ms para la mayoria de casos UI.
  • Combina animation para secuencias automaticas y transition para interaccion con hover/focus.
