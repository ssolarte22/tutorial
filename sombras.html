Sombras de <div> (cajas)

Con CSS3 podemos crear sombras para zonas del HTML (<div>, <article>, <footer>, <header>...) y también para textos. Para cajas se usa la propiedad box-shadow.

Sintaxis básica:
box-shadow: 6px 20px 10px black;

  • El primer valor indica el desplazamiento horizontal.
  • El segundo valor indica el desplazamiento vertical.
  • El tercer valor define la suavidad de la sombra.
  • El último valor corresponde al color.

Sombras internas

Para crear una sombra interna se añade el valor inset al final:
box-shadow: 7px 20px 10px black inset;

Creación de halos o resplandores

Para crear un halo alrededor de una caja se usan 0px en los dos primeros valores y se define el color del resplandor:
box-shadow: 0px 0px 30px blue;

Sombras en textos

Para textos se utiliza text-shadow, con la misma lógica que box-shadow:
text-shadow: 10px 10px 5px gray;

  • Primer valor: separación horizontal de la sombra.
  • Segundo valor: separación vertical de la sombra.
  • Tercer valor: suavidad de la sombra.
  • Último valor: color de la sombra.

Múltiples sombras en el mismo elemento

Para aplicar varias sombras en un mismo elemento, se separan con comas:
box-shadow: 0px 0px 20px red inset, 0px 0px 25px blue;

En este ejemplo, la primera sombra es interna (inset) y la segunda es externa.
