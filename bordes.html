Líneas separadoras horizontales

Para crear una línea horizontal, la forma más rápida es usar la etiqueta <hr>, que no se cierra y genera una línea separadora simple.

Esta etiqueta puede personalizarse con CSS usando propiedades como: border-width (grosor), border-style (tipo), border-color (color) y width (anchura).

Líneas separadoras verticales

Para crear líneas verticales se define uno o varios bordes de una caja <div> con estas propiedades:
border-top: 1px solid red;
border-bottom: 2px dotted green;
border-right: 1px dashed #3C5391;
border-left: 1px inset orange;

Bordes simples

Para indicar un borde en un <div> se usa la propiedad border con este formato: border: 2px solid grey;

Tipos de borde disponibles:
none: sin línea.
solid: línea continua.
dotted: línea punteada.
dashed: línea discontinua.
inset: efecto de bajorrelieve.
outset: efecto de relieve.

Borde simple con tamaño definido y esquinas redondeadas.

width: 300px;
height: 100px;
border: 3px solid rgb(red, green, blue);
border-radius: 22px;
padding-left: 15px;
padding-top: 30px;

Bordes redondeados

Con border-radius se pueden redondear las esquinas de un <div>, aplicando un valor único para las cuatro esquinas o valores distintos para cada una.

Para que se aprecie el redondeo, es necesario definir también un borde normal (border) y/o un color de fondo (background-color).

Orden de los cuatro valores de border-radius: superior-izquierda, superior-derecha, inferior-derecha, inferior-izquierda.

Ejemplo

Ejemplo de borde con valores diferentes.
border-radius: 10px 0px 39px 22px;

Ejemplo práctico

El ejemplo completo combina: width y height para el tamaño del <div>, border para el borde, border-radius para suavizar esquinas, background-color para el fondo y padding para separar el texto del borde.
