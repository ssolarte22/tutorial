Transformaciones CSS (transform)

1) ¿Qué es transform?

La propiedad transform de CSS3 te permite modificar el espacio de coordenadas de un elemento para trasladarlo (moverlo), rotarlo, escalarlo, o sesgarlo (deformarlo). A diferencia de cambiar el margin o top/left, las transformaciones están aceleradas por hardware (GPU) y no afectan el flujo o la posición de los elementos adyacentes.

2) Las funciones principales

  • translate(x, y): Mueve el elemento por los ejes horizontal y vertical. (Ej. translate(50px, 100px))
  • scale(x, y): Altera el tamaño del elemento. El 1 es el tamaño original. (Ej. scale(1.5) aumenta un 50%)
  • rotate(angle): Gira el elemento según los grados indicados. (Ej. rotate(45deg))
  • skew(x-angle, y-angle): Inclina el elemento. (Ej. skew(20deg, 10deg))

4) Transform-origin

Por defecto, el punto desde el cual se calcula la transformación es el centro exacto del elemento (`50% 50%`). Si deseas que, por ejemplo, un elemento rote sujetado desde la esquina superior izquierda, utilizarías:
transform-origin: top left;
