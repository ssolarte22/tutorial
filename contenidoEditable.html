Contenido Editable (contenteditable)

1) ¿Qué es contenteditable?

El atributo global contenteditable en HTML5 permite a los usuarios modificar el texto de los elementos HTML directamente desde la página, sin necesidad de usar inputs heredados como <textarea>.

Basta con añadir contenteditable="true" a casi cualquier etiqueta (div, p, span, li) para convertirla en una zona editable al estilo de un procesador de textos.

2) Demo: Bloque editable interactivo

4) Aplicaciones reales

  • Editores WYSIWYG: Es la tecnología base para creadores de contenido (como WordPress, editores de correos, etc.).
  • Edición In-line: Permite a los usuarios actualizar datos de su perfil o configuraciones haciendo clic directo, mejorando la experiencia de usuario.
  • Para guardar los cambios, se requiere JavaScript. Escuchando los eventos blur o input, puedes recolectar el HTML interno del bloque y mandarlo a tu servidor o guardarlo en Storage.
