Imágenes de fondo

Las palabras mágicas

En HTML5 podemos añadir fondos con un color sólido, un color semitransparente, una imagen o varias imágenes superpuestas.

Color de fondo

Para aplicar un color de fondo a toda la página o a un <div> se usa: background-color.

Color de fondo semitransparente

Con rgba(r, g, b, a) el cuarto valor es la opacidad: 0 = totalmente transparente y 1 = totalmente opaco.

Imagen de fondo

Para colocar una imagen se usa background con url(...). Si no se indica nada más, la imagen se repite en mosaico. También es importante recordar que la ruta depende de dónde está el CSS.

Repetición de la imagen

background-repeat permite controlar la repetición:
no-repeat: no se repite.
repeat-x: se repite en horizontal.
repeat-y: se repite en vertical.

Tamaño de la imagen

background-size permite ajustar dimensiones en píxeles o porcentaje, por ejemplo: 200px 100px o 25%.

Superposición de múltiples imágenes como fondo

En CSS3 se pueden apilar varias imágenes en un mismo fondo separándolas por comas. El orden importa: la primera imagen queda por encima y la última por debajo.

Esta técnica funciona mejor cuando las capas superiores tienen zonas transparentes, para permitir ver las imágenes inferiores.

Extras

Fondo fijo

background-attachment: fixed permite que el fondo no se mueva al hacer scroll.

Fondo adaptable al contenedor

Para adaptar un fondo al tamaño del contenedor se suele usar: background-repeat: no-repeat y background-size: cover.
