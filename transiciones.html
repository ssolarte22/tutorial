Transiciones (transition)

1) ¿Qué es una transición de CSS?

Las transiciones permiten que los cambios en los valores de las propiedades CSS ocurran sin problemas durante una duración especificada, en lugar de ocurrir de manera instantánea. Suelen utilizarse junto con pseudo-clases como :hover y :focus para crear interfaces interactivas.

2) Sintaxis y Propiedades

Para crear una transición, necesitas especificar al menos dos cosas:

  • transition-property: El nombre de la propiedad CSS a la que deseas aplicar el efecto (ej: background-color, width, o all).
  • transition-duration: La duración del efecto (ej: 2s o 500ms).

Atributos opcionales muy útiles:

  • transition-timing-function: Controla la aceleración de la transición (ease, linear, ease-in-out).
  • transition-delay: Retraso antes de que comience la transición.
/* Shorthand o forma corta: */
.elemento {
  transition: all 0.3s ease-in-out;
}
