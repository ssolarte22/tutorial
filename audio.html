Audio en HTML5

1) Objetivo del tema

HTML5 permite insertar audio sin plugins externos mediante la etiqueta <audio>. El navegador crea un reproductor nativo con controles de reproduccion, pausa y volumen.

Lo mas recomendable es ofrecer varias fuentes para mejorar compatibilidad entre navegadores.

2) Estructura basica

<audio controls>
  <source src="audio/tema.mp3" type="audio/mpeg">
  <source src="audio/tema.ogg" type="audio/ogg">
  Tu navegador no soporta audio HTML5.
</audio>

La etiqueta <source> permite definir formatos alternativos. Si uno falla, el navegador intenta el siguiente.

3) Demo con controles

Este reproductor incluye controles nativos. Si no existe el archivo todavia, aparecera el reproductor, pero no sonara hasta que agregues un audio valido.

4) Atributos mas usados

  • controls: muestra botonera del reproductor.
  • autoplay: reproduce automaticamente (puede bloquearse segun politicas del navegador).
  • loop: repite el audio al finalizar.
  • muted: inicia en silencio.
  • preload: sugiere como cargar contenido (none, metadata, auto).

5) Demo de playlist simple (estructura)

Puedes combinar una lista de pistas con un reproductor unico. Aqui tienes la estructura base: 

<ul>
  <li><button>Pista 1</button></li>
  <li><button>Pista 2</button></li>
</ul>
<audio id="player" controls></audio>

Con JavaScript puedes cambiar el atributo src del reproductor al pulsar cada boton.

6) Buenas practicas

  • No uses autoplay con sonido por accesibilidad y experiencia de usuario.
  • Muestra siempre texto alternativo dentro de <audio> para navegadores no compatibles.
  • Optimiza peso de los archivos para carga rapida.
  • Usa nombres y carpetas claras, por ejemplo: media/audio/intro.mp3.
