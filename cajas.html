Cajas flotantes (float)

Uso de float

La propiedad float sirve para alinear cajas o <div> a la izquierda o a la derecha. Los valores más usados son float:left y float:right.

Para alinear varias cajas una junto a otra, todas deben tener un width definido y usar float. Además, la suma de sus anchos debe caber en el contenedor.

Si después de cajas flotantes se añade otro elemento, este puede ocupar el hueco libre. Para evitarlo se usa clear:both en la siguiente caja.

Reglas importantes

  • Todas las cajas alineadas deben tener width + float.
  • Si no caben en el ancho disponible, bajarán a la siguiente línea.
  • Normalmente se usa la misma dirección de float para el grupo.
  • clear:both evita que un elemento se desplace al hueco libre.
  • Si quieres colocar dos cajas seguidas, normalmente se usa float:left en ambas.

Ejemplo práctico del uso de float

En este ejemplo se usan dos cajas flotantes una al lado de la otra, con width definido y float:left.

Codigo CSS


                #caja1{
                    width:200px;
                    float:left;
                    margin-right:15px;
                }
                #caja2{
                    width:200px;
                    float:left;
                }

Codigo HTML


                <div id="cajas">
                    <div id="caja1">
                    </div>

                    <div id="caja2">
                    </div>
                </div>

Alternativa con contenedor

Otra solucion elegante es meter las cajas flotantes dentro de un contenedor con altura definida.


Codigo HTML


                <div id="cajas">
                    <div id="caja1">
                </div>

                <div id="caja2">
                    </div>
                </div>

Codigo CSS


                #cajas{
                height:131px;
                }
                
                #caja1{
                width:395px;
                float:left;
                margin-right:15px;
                }

                #caja2{
                width:300px;
                float:left;
                }
